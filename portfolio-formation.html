PizzaMama

Avec Django

J'ai réalisé ce projet en février 2022 pour m'entrainer faire un site avec un framework.
Pour PizzaMama, j'ai utilisé django sous python avec une base de donnée SQL.
j'ai hebergé ce site sur Heroku. J'ai acquis les capacités de produire ce site web grâce à la formation de Jonathan Roux sur Udemy.
Cliquez sur l'image pour pouvoir le visiter !

Consulter le projet

Etant produit avec django, j'ai pu rendre ce site modulable.
Il y a une partie administateur dans laquelle je peux ajouter, modifier, enlever des pizzas, bref, faire tous les changements nécessaires.
Les Pizzas sont stockées dans une base SQL qui se met à jour à chaque modification.
